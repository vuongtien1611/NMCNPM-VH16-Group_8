# NMCNPM_VH16_Group8
**ShopAdmin – Hệ Thống Quản Trị TMĐT (Group 8)**

📌 Giới thiệu dự án
ShopAdmin là hệ thống quản trị thương mại điện tử được phát triển bởi Group 8 (VH16). Hệ thống cung cấp giải pháp quản lý toàn diện cho cửa hàng, tích hợp API bảo mật thực hiện CRUD thời gian thực trên sản phẩm, đơn hàng và khách hàng.

## 🚀 Tính năng chính

### 📊 1. Dashboard (Tổng quan)
- Hiển thị tổng doanh thu
- Tổng số đơn hàng
- Lợi nhuận
- Khách hàng mới
- Biểu đồ doanh thu 7 ngày gần nhất
- Biểu đồ cơ cấu sản phẩm

### 📦 2. Quản lý sản phẩm
- Thêm sản phẩm
- Sửa sản phẩm
- Xóa sản phẩm
- Tìm kiếm sản phẩm
- Lưu trữ dữ liệu bằng LocalStorage

### 🧾 3. Quản lý đơn hàng
- Tạo đơn hàng mới
- Cập nhật trạng thái đơn hàng
- Xem chi tiết đơn
- Tính tổng tiền tự động

### 👥 4. Quản lý khách hàng
- Thêm khách hàng
- Cập nhật thông tin
- Tìm kiếm khách hàng
- Thống kê khách hàng mới

### 📈 5. Báo cáo doanh thu
- Lọc theo ngày
- Thống kê doanh thu
- Thống kê lợi nhuận
- Biểu đồ trực quan

## 🛠 Công nghệ sử dụng

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
- Chart.js
